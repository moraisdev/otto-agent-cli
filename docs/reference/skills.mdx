# Skills

`otto skills` installs, lists and inspects Otto skills.

The default source is the official Otto skill catalog generated from:

```text
src/plugins/internal/**/skills/*/SKILL.md
```

External sources are explicit and use `--source`. A source can be a GitHub shorthand, GitHub URL, git URL, or local path.

## List Catalog Skills

```bash
otto skills list
otto skills list --json
```

## List Installed Skills

```bash
otto skills list --installed
otto skills list --installed --codex --json
```

## List Skills In A Source

```bash
otto skills list --source vercel-labs/skills
otto skills list --source https://github.com/vercel-labs/skills/tree/main/skills/find-skills
```

## Show A Skill

```bash
otto skills show image
otto skills show image --installed
otto skills show find-skills --source vercel-labs/skills
```

## Install

```bash
otto skills install image
otto skills install --all
otto skills install find-skills --source vercel-labs/skills
otto skills install --source ./my-skills --all
```

If a catalog/source contains multiple skills, installation is explicit: pass a skill name or `--all`.

Installed operator skills live in:

```text
~/otto/plugins/otto-user-skills/skills/<skill>
```

After install, the CLI syncs materialized Codex skills automatically unless `--skip-codex-sync` is passed.

## Sync

```bash
otto skills sync
```

Use this when plugin skill files changed on disk and you want the materialized skills directory refreshed immediately.
